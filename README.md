# Erinc Argimak - CV

## Development

```bash
npm install
npm run dev # -- --open # open app in a new browser tab
```

## Building

Build a production version:

```bash
npm run build
npm run preview
```

> To deploy your app, you may need to install an [adapter](https://kit.svelte.dev/docs/adapters) for your target environment.
